# As contribuições positivas da inteligência artifical (IA) para as inovações no mercado de trabalho.

## 📒 Descrição
O ebook explana as mudanças no mercado de trabalho eminente orientado para novas tecnologias e inovações, em especial a Inteligencia Artificial (IA), desenvolvido com referência no Manual de Oslo 4ª Edição e os conteúdos do Serviço de Apoio às Micro e Pequenas Empresas (Sebrae).

## 🤖 Tecnologias Utilizadas
Ferramentas usadas:
1. Site: ChatGPT 4o mini - criação do ebook formato texto
2. Site: Gamma - diagramação do ebook

## 🧐 Processo de Criação
1. Desenvolvi o prompt para que a IA generativa do ChatGPT desenvolvesse para mim um ebook sobre o impacto da ia no mercado de trabalho e especifiquei a referência como o Manual de Oslo e conteúdos do Sebrae.
2. Copiei o texto gerado dentro do site Gamma para desenvolvimento de apresentação com uso de IA, após a geração do slide, conferi e atualizei/corrigi alguns slides e conteúdos suprimidos.
